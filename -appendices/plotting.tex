As mentioned by wikipedia\footnote{\url{https://en.wikipedia.org/wiki/Utopia_(typeface)}},
Utopia can be downloaded in {*}.otf, {*}.ttf and other formats for
free:
\begin{itemize}
\item \url{https://ghostscript.com/~tor/stuff/fonts/utopia/} or
\item \url{https://code.google.com/archive/p/evristika/}.
\end{itemize}
These fonts can be read directly by matplotlib, but cannot be used
in mathtext. The font Stix blends in decently, but better (though
slower) is using latex to typeset math in matplotlib. Comments in
the matplotlib style file show how this could be done.
