This is the TU Delft dissertation template for \LaTeX. The source files can be found at GitLab\footnote{\url{https://gitlab.com/novanext/tudelft-dissertation}}. It is designed to work with pdf\LaTeX\ and Lua\LaTeX, and produce output adhering to the TU Delft corporate design style\footnote{\url{https://www.tudelft.nl/en/tu-delft-corporate-design}}.

The template can be used with Overleaf and supports the combination of Arial\footnote{\url{https://en.wikipedia.org/wiki/Arial}} with Roboto Slab\footnote{\url{https://fonts.google.com/specimen/Roboto+Slab}
} or Roboto\footnote{\url{https://fonts.google.com/specimen/Roboto}} with Roboto Slab. Besides, the font Utopia\footnote{\url{https://fonts.adobe.com/fonts/utopia}} using the \LaTeX\ package Fourier\footnote{\url{https://ctan.org/pkg/fourier}}.
